어휘의 힘

가장 비싼 도구는 사전입니다

AI는 내가 아는 만큼 대답합니다.

같은 AI, 같은 화면. 결과가 다른 이유는 '내가 쓴 단어' 하나입니다.

❌ 그냥 풀어서 설명했을 때
"이 숫자들 중에서 다른 값들에 비해 너무 크거나 작아서 평균이나 일반적인 흐름에서 크게 벗어나는 것들만 골라서 보여줘. 너무 많이 벗어난 거 위주로…"
→ '많이'의 기준을 AI가 임의로 설정 → 결과 들쭉날쭉
✅ 개념어 1단어로 설명했을 때
"이상치 탐지해줘"
→ IQR · Z-score · 표준편차 기반의 통계적 정의 자동 활성
Anthropic 공식 프롬프팅 가이드의 첫 줄 — "Use specific, domain-relevant terminology."
AI를 만든 회사가 직접 말합니다: 구체적인 전문 용어를 쓰세요.

🆚 Code 앱 vs Code CLI — 가장 큰 차이는?
🖥 Code 앱 = "보기 편함"
변경 전후를 Diff 뷰로 한눈에 · 결과물 실시간 미리보기 · 마우스 클릭 위주 · 한 번에 한 작업
💻 Code CLI = "자동화 강함"
스크립트로 묶어 매일 자동 실행 가능 · Dooray 같은 외부 서비스 연동 쉬움 · 밤새 돌리고 결과만 받기 OK
→ 한 번씩 손으로 확인하며 작업할 거면 앱, "매주/매월 자동으로 돌리고 싶다"면 CLI. 오늘은 CLI를 다룹니다.

Dooray MCP 3단계 설치

한 번 설치, 다음부터는 자동 연결

1
Dooray API 토큰 발급
Dooray 우상단 My → 인증/Token → "신규 토큰 발급"
토큰은 한 번만 표시 — 바로 복사
2
환경변수 등록
# Mac $ export DOORAY_API_TOKEN="xxxxxx" # Windows PowerShell $ $env:DOORAY_API_TOKEN="xxxxxx"
3
MCP 등록
$ claude mcp add dooray-mcp \ "npx -y dooray-mcp"
→ /mcp로 정상 연결 확인
Dooray MCP 실습

한국어 5문장이면 충분합니다

1. 조회
$ "내 두레이 태스크 보여줘"
목록 출력되면 성공
2. 필터
$ "오늘 마감인 태스크만 필터해줘"
기한 필터 자동 적용

$ "'주간보고 작성' 태스크를 김래영 선임에게 내일 마감으로 생성해줘"
웹에서 새 태스크 보이면 ✅
4. 댓글
$ "방금 만든 태스크에 '검토 완료' 댓글 달아줘"

5. 상태 변경
$ "이 태스크 상태를 '진행 중'으로 바꿔줘"

슬래시 명령어

가장 많이 쓰는 슬래시 명령어 9종

#명령어용도비개발자 언어
1/usage토큰 사용량 확인"내가 얼마나 썼지?" → 요금 폭탄 방지
2/clear대화 리셋"새 주제 시작할 때" → 토큰 90% 절약
3/resume이전 대화 이어하기"터미널 닫았다 다시 열었을 때"
4/cost현재 세션 비용"이번 작업 얼마짜리였지?"
5/doctor환경 진단"뭔가 이상할 때 제일 먼저 실행"
6/help전체 명령어 목록"막히면 무조건 여기"
7/initCLAUDE.md 자동 생성"새 프로젝트 시작할 때 규칙집 만들기"
8/review작업 검토 요청"방금 한 작업 한 번 더 봐줘"
9/memory기억 파일 확인"Claude가 나에 대해 뭘 기억하고 있지?"
9개만 알면 거의 다 됩니다. 처음엔 1~3번(/usage /clear /resume)부터 손에 익히세요.
